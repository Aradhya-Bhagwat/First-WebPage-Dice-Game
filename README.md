# Roll the Dice Game

A simple, interactive 2-player dice game built using HTML, CSS, and JavaScript.

## How to Clone

To clone this repository to your local machine, open your terminal and run one of the following commands:

### Using HTTPS
```bash
git clone https://github.com/Aradhya-Bhagwat/First-WebPage-Dice-Game.git
```

### Using SSH
```bash
git clone git@github.com:Aradhya-Bhagwat/First-WebPage-Dice-Game.git
```

---

## How to Run

After cloning, you can run the game by following these steps:

1. Navigate to the project directory:
   ```bash
   cd First-WebPage-Dice-Game
   ```
2. Open the `index.html` file in any modern web browser:
   - On Windows (PowerShell):
     ```powershell
     Start-Process "index.html"
     ```
   - On macOS:
     ```bash
     open index.html
     ```
   - On Linux:
     ```bash
     xdg-open index.html
     ```
   - Or simply double-click the `index.html` file in your file explorer.
